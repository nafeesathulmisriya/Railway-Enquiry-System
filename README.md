Railway Enquiry and Ticket Booking System

The Railway Enquiry and Ticket Booking System is a web-based application developed using Python and Django that allows users to search for trains, check schedules, book railway tickets, and complete payments online.
The system provides a simple and efficient platform for passengers to access railway information and make reservations without visiting railway counters.

🎯 Project Objectives

Provide an online platform for railway enquiry and ticket booking.
Allow users to search trains and view schedule details.
Enable passengers to book tickets and complete payments digitally.
Reduce manual workload at railway reservation counters.
Improve accessibility and convenience for passengers.

⚙️ System Modules
👨‍💼 Admin Module
The admin manages the system through the Django admin panel.

Features:
Add, update, and delete train details
Manage train schedules
View and manage ticket bookings
Monitor user activities
Manage payment records

👤 User Module
Passengers can access railway information and book tickets.

Features:
User registration and login
Search trains between stations
View train schedules and details
Book train tickets
Make online payments
View booking history

✨ Key Features

Train search functionality
Train schedule display
Ticket booking system
Online payment integration
Booking confirmation
User-friendly interface

📊 System Workflow

User registers or logs into the system.
User searches for trains between source and destination.
Available trains and schedules are displayed.
User selects a train and books a ticket.
User completes payment through the payment system.
Booking confirmation is generated and stored in the database.
